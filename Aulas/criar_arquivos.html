Criando e salvando arquivos no repositório

Comandos importantes

1. git status

Mostra os arquivos que faltam ser salvos no repositórios, ou commits, etc.

2. git add [nome do arquivo]

Adiciona arquivos modificados ou novos em um pacote (commit).

Você pode adicionar todos os arquivos que ainda não foram empacotados usando o comando git add . (com o ponto final mesmo).

3. git commit -m"[Mensagem do commit]"

Empacota as modificações e registra, recebendo uma mensagem de identificação dentro das aspas.

4. git push

Envia o pacote de modificações para o repositório

Outros Comandos

1. touch

Cria arquivos. Ex: touch README.md, que a professora usou para criar um arquivo readme na pasta do repositório. Clique aqui para acessar um site que lhe ajuda a criar arquivos markdown com design bonito.

2. rm -rf [nome da pasta]

Deleta pastas e todo seu conteúdo.

3. mkdir [nome da pasta]

Cria pastas

4. git log

Mostra informações sobre o último comando passado no terminal.

5. Arquivos .gitignore
